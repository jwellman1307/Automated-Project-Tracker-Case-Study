# Automated Project Tracker – Excel Workbook Case Study

The **Automated Project Tracker** is an Excel-based project management workbook built to help users organize projects, track tasks, document release versions, and keep development notes in one structured place.

This repository is a case study and documentation showcase for the workbook, including screenshots, a demo video, feature documentation, and development notes.

---

## Project Overview

The Automated Project Tracker was designed for creators, developers, freelancers, students, and small teams who need a simple but structured way to manage projects without using a full project management platform.

The workbook focuses on:

- Creating and tracking multiple projects
- Managing project tasks
- Recording version releases
- Saving release notes
- Keeping brainstorming and development notes
- Reviewing project progress from a central dashboard

The goal was to build a practical Excel tool that feels more like a lightweight project app than a basic spreadsheet.
